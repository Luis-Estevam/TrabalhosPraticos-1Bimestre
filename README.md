# Projetos da materia de Estrutura, Pesquisa e Ordenação de Dados
Repositorio separado entre os 3 projetos do primeiro bimestre da materia de Estrutura, Pesquisa e Ordenação de dados.
# Arquitetura do Projeto
```bash
Trabalhos/
│
├── ProjetoSorts/        # Algoritmos de ordenação e benchmark
├── ProjetoBuscas/       # Algoritmos de busca e análise estatística
├── ProjetoArvores/      # Estruturas de dados e TSP
└── Main.java            # Ponto de entrada com menu interativo
```
# Como executar
```bash
git clone https://github.com/Luis-Estevam/TrabalhosPraticos-1Bimestre.git
cd TrabalhosPraticos-1Bimestre
javac Main.java
java Main
```
